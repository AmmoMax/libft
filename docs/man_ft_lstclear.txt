Функция ft_lstclear()

void	ft_lstclear(t_list **lst, void (*del)(void *));

Параметры:
lst - адрес указателя на массив со структурами, т.е. по сути указатель на весь большой список
del - указатель на функцию, которая очищает content каждого элемента

Возвращаемое значение:
нет

Разрешенные фукнции:
free

Описание:
Очищает весь список удаляя содержимое content каждого элемента и очищая память, которую занимает каждый член структуры(элемент списка)
В конце указатель на список должен быть присвоен NULL
