Функция ft_itoa() - конвертирует число в строчное представление.

char	*ft_itoa(int n);

Аргументы:
n - число для конвертации

Возвращаемое значение:
Строка представляющее целое число. NULL - если память не выделилась.

Описание:
Выделяет память и возвращает строку, представлющую целое число, переданное как аргумент.
Отрицательные числа должны быть обработаны.
