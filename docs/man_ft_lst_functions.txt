Общий ман по всем бонусным функциям, работающим со структурами.

typedef struct		s_list
{
	void			*content;
	struct s_list	*next;
}					t_list;

content - данные содержащиеся в элементе структуры. Указатель на тип void позволяет хранить любой тип данных
next - адрес следующего элемента структуры или нулевой указатель (NULL) если это последний элемент.(имеется в виду реализация односвязного списка)
