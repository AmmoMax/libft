ft_memset - Заполняет массив указанными символами. Возвращает указатель на заполненный массив.
ft_bzero - Записывает нули в область памяти указанной длины. Ничего не возвращает.
ft_memcpy - Копирует указанное количество байт из одной области памяти в другую. Возвращает указатель на новую область памяти.
ft_memccpy - Копирует указанно количество байт из одной области памяти в другую и останавливается когда найден символ, переданый как аргумент. Возвращает указатель на следующий символ в destination после символа c. Если символ c не был найден возвращает NULL.
ft_memmove - Копирует указанное количесво символов из одной области памяти в другую, но делает это через временный массив, обеспечивая как бы "безопасность". Возвращает указатель на область памяти - получатель.
ft_memchr - Находит первое вхождение символа, переданного как аргумент, в указанном количестве начальных символов области памяти. Возвращает указатель на найденный символ. Если символ не найден - NULL.
ft_memcmp - Сравнивает побайтно две области памяти на длину, определенную как аргумент. Сравнение продолжается до тех пор, пока не будет проверено определенное количество байт или пока не встретятся отличающиеся байты. Возвращает 0, если области памяти одинаковые или указанная длина поиска нулевая. Для ненулевого возвращаемого значения знак определяется знаком разности между первой парой байт(интерпретируемых как unsigned char), которые отличаются в s1 и s2.
ft_strlen - Считает длину строки, исключая символ конца строки \0.
ft_strlcpy - BSD функция. Копирует нуль-терминированную строку из одной области памяти в другую, количество символов для копирования определяется параметром. Строка назначения так же нуль-терминируется. Возвращает общую длину строки, которую пыталась скопировать.
ft_strlcat - BSD функция. Добавляет одну нуль-терминированную src строку в конец строки dst. Результат нуль-терминируется. Возвращает общую длину строки, с которой пыталась работать - в идеале длину dst + src.
ft_strchr - Находит первое вхождение указанного символа в строке, завершающий символ считается частью строки. Возвращает указатель на найденный символ. Если символ не найден - возвращает NULL.
ft_strrchr - Находит последнее вхождение указанного символа в строку. Возвращает указатель на последний символ в строке. Если символ не найден - NULL.
ft_strnstr - Ищет строку в подстроке.
ft_strncmp - сравнивает посимвольно строку s1 и строку s2 на длину определенную как третий аргумент функции. Возвращает 0 если длина 0, 0 если строки одинаковы и разность двух отличающихся символов в строках.
