Функция ft_substr

char *ft_substr(char const *s, unsigned int start, size_t len);

s - строка, из которой будет создаваться подстрока
start - индекс символа в строке s, с которого будет создаваться подстрока
len - максимальная длина подстроки

Возвращаемое значение:
Указатель на созданную подстроку. В случае ошибки выделения памяти - NULL.

Выделение памяти с помощью функции malloc() и возврат подстроки строки s. Подстрока начинается с индекса start и максимальная длина len.

Над какими кейсами стоит подумать при реализации:
1. Защита в случае если start > len(s)
