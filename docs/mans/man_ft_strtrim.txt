char	*ft_strtrim(char const *s1, char const *set);

s1 - исходная строка, из которой надо удалить символы
set - эталонный набор символов для обрезки строки s1

Возвращаемое значение:
Строка с удаленными символами из набора set. Если выделение памяти прошло неудачно вернуть NULL.

Описание:
Выделение памяти и возврат копии строки s1 c удаленными символами из набора set в начале и конце строки.


Функция ft_strtrim() находит первое вхождение символов из set в исходной строке от начала и с конца и отрезает этот кусок.
Если первый символ исходной строки не входит в set - с начала ничего не обрезаем.
Если первый символ с конца не входит в set - также ничего не обрезаем.


Примерный алгоритм работы:
1. Берем первый символ с начала строки и проверяем входит ли он в set.
	1.1 Да. Берем второй символ и снова проверяем входит ли он в set.
	1.2 Нет. Запоминаем его позицию - это место до которого мы обрежем строку с начала.
2. Берем последний символ в строке и проверяем входит ли он в set.
	2.1 Да. Берем следующий символ и снова проверяем.
	2.2 Нет. Запоминаем позицию - это место до которого мы обрежем строку с начала.

3. Зная позиции начального и конечного символа рассчитываем длину + 1 и выделяем память.
4. В цикле еще раз пробегаемся по исходной строке и записываем в результирующую строку.
5. Возвращаем результат.

Дополнительные условия:
1. Если входящие строки - нулевой указатель - возвращаем NULL
2. Если первый и последний символ исходной строки не входят в заданный массив - копируем и возвращаем исходную строку.
3. Контролируем выделение памяти - если память выделяется с ошибкой - возвращаем NULL.
