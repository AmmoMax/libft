Функция ft_lstiter()

void	ft_lstiter(t_list *lst, void (*f)(void *));

Параметры:
lst - указатель на элемент списка.
f - указатель на функцию, которая используется при итерации по списку.

Возвращаемое значение:
Нет

Разрешенные функции:
Нет

Описание:
Итерируемся по списку и применяем функцию f к полю content каждого элемента.
