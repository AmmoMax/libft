Функция bzero - записывает пустые байты в строку

#include <string.h>
void	bzero(void *s, size_t n);

Функция bzero удаляет данные в n байтах памяти, начиная с места, куда указывает s, записывая нули (\0) в эту область памяти.

Входные данные для тестов:
Нулевой указатель
Положительный размер
Отрицательный размер
Нулевой размер
Указатель на несуществующую область памяти
