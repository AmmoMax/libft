Функция strlcat - нестандартная функция для объединения строк.

size_t	strlcat(char *dst, const char *src, size_t size);

Функция strlcat() добавляет нуль-терминированную строку src в конец строки dst. Она будет добавлять не более (size - strlen(dst) -1) байт. Результат будет нуль-терминирован.

Возвращаемое значение:
Возвращает общую длину строки, с которой пыталась работать, т.е. начальную длину dst + длину src.

Параметр size - это buffer, это предельно возможное место под результат.

====
Различные варианты входных данных, которые необходимо реализовать.

1. Если размер size-1 меньше чем len(dst) - конкатенация не выполняется, возвращаем size+len(dst)
2. Если размера size не хватает чтобы вместить dst+src+\0 - тогда умещается то, что влезает и возвращается сумма длин строк.
