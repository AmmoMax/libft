Функция memset - заполнение массива указанными символами.

Синтаксис:
#include <string.h>
void	*memset(void *destnation, int c, size_t n)

Значения аргументов:
destnation - указатель на заполняемый массив
с - код символа для заполнения
n - размер заполняемой части в байтах

Возвращаемое значение:
Функция возвращает указатель на заполняемый массив.

Описание:
Фукнция memset заполняет первые n в области памяти, на которую указывает
 аргумент destination, символом, код которого указывается аргументом c.

 =====

 Примерный алгоритм работы моей функции:

 Итерируемся по полученному массиву
  и на каждой итерации записываем в текущую позицию символ из аргументов.
  Количество итераций определяется аргументом n.
