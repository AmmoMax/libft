Функция memccpy копирует область памяти.

#include <string.h>
void	*memccpy(void *dest, const void *src, int c, size_t n);

Функция копирует не больше чем n байт из области памяти src в област памяти dest и останавливается когда найден символ c.

Если области памяти пересекаются - результат будет неопределеным.

Возвращает указатель на следующий символ в dest после c или NULL(нулевой указатель) если c не был найден в первых n символах src.
