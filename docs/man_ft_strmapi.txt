Функция ft_strmapi()

char	*ft_strmapi(char const *s, char (*f)(unsigned int, char));

Параметры:
s - строка, по который мы будем итерироваться
f - функция для применения к каждому символу

Возвращаемое значение:
Строка созданная из последовательных ответ функции f. NULL - если память не удалось выделить.
