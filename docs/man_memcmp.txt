Функция memcmp побайтно сравнивает два массива (области памяти), на которые указывают аргументы s1 и s2. 
Каждый байт массива определяется типом unsigned char.
Сравнение продолжается, пока не будут проверено n байт или пока не встретятся отличающиеся байты. 


# include <string.h>

int memcmp(const void s1, const void s2, size_t n);


Функция сравнивает первые n байт (каждый интерпретируется как unsigned char) областей памяти s1 и s2.

Возвращаемое значение:

Число меньше нуля, если первые n байт области s1 меньше чем первые n байт области s2.
Число больше нуля, если первые n байт области s1, больше чем первые n байт области s2.
Ноль, если первые n байт области s1 совпадают с n байтами области s2.

Для ненулевого возвращаемого значения знак определяется знаком разности между первой парой байт(интерпретируемых как unsigned char), которые отличаются в s1 и s2.

Если n == 0, возвращается ноль.
